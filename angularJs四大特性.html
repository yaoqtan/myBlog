Angular四大核心特性

  • MVC

    			

<body ng-app ="mymodule">
	<div ng-controller = "myDemo">
		<p>{{name}}</p>
	</div>
	<script type="text/javascript">
		var APP = angular.module("mymodule",[]);
		APP.controller('myDemo',['$scope',
			function($scope){
				$scope.name = "name"
			}
		]);
	</script>
</body>

    上例代码中，P标签充当V(视图)，mymodule函数为C(控制器)，$scope.name = "name"为M(数据模型)

    使用 $cope 实现依赖注入

  • 模块化

    var APP = angular.module("mymodule",[]); 使用angular.module()方法定义模块

    一切都是从模块开始

  • 指令

    <body ng-app="myApp"> <directive-hello></directive-hello> <script> var app = angular.module("myApp", []); app.directive("directiveHello", function() { return { restrict:'E', template : "<h1>自定义指令!</h1>", replace:true }; }); </script> </body>

    restrict：通过什么方法调用指令 (默认EA)

    • E 作为元素名使用:<runoob-directive></runoob-directive>
    • A 作为属性使用:<div runoob-directive></div>
    • C 作为类名使用:<div class="runoob-directive"></div>
    • M 作为注释使用:<!- - directive: runoob-directive - ->

  • 双向数据绑定

    视图和数据是对应的，当视图发生变化，数据模型会发生变化；当数据模型发生变化，视图自动更新。

    在html中，html中发生变化的视图：表单。当表单发生变化，数据可以同步数据上

    更改$scope.name的值，input标签值改变；改变input值，p标签的是值也跟着改变；从而实现双向数据绑定

    			
				
<body ng-app ="mymodule">
	<div ng-controller = "myDemo">
	<input type="text" name="text" ng-model = "name" value = "test" >
	<p>{{name}}</p>
	</div>
	<script type="text/javascript">
		var APP = angular.module("mymodule",[]);
		APP.controller('myDemo',['$scope',
			function($scope){
				$scope.name = "name"
			}
		]);
	</script>
</body>
